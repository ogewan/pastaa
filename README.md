# pastaa
Retrieve script via url on Node.js.

## Installation
* yarn add pasta

* npm install pasta*

## Usage
```js
let pastaa = require('pastaa');
    //returns the context that the script is run in.
    jQuery = await pastaa('https://code.jquery.com/jquery-3.3.1.slim.min.js'),
    $ = jQuery.$;
```
